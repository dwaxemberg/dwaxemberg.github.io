About Me

Since my first programming class in high school, I have been in love with software engineering. I have worked on several personal and professional projects. Through these projects I have been exposed to several programming languages and a vast array of platforms and frameworks. From embedded systems to mobile applications, I have been exposed to all sorts of different levels of programming. By serving multiple internships I have gained knowledge on the quality of code required for professional software development.

While I have had several experiences with software engineering, there are still many skills I would like to learn and improve. I am most actively pursuing learning the programming language: Swift. Another one of my goals is to continue working on one of the Android and iOS applications I started working on last year with my roommates: Limelight Game Streaming. I am constantly learning skills from my friends, coworkers and classes that I hope to apply to future projects.

After I graduate on the sixteenth of December 2014 from Case Western Reserve University I have a full time job as a software engineer from Yelp, Inc. where I will be on the Android Application team. My responsibilities include designing new features for the application as well as supporting legacy code and fixing software bugs that users report. Another one of my duties will be reviewing my team members' code before it is pushed to production.
